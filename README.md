# BaobabX - Crypto Trading Dashboard

## 주요 기능

- 실시간 캔들스틱 차트
- 실시간 오더북 (20호가)
- 시장 정보 (가격, 24h 변동률, 거래량)
- 거래쌍 선택 (BTC, ETH, SOL / PERP, SPOT)
- 포지션 및 주문 관리 패널

## 시작하기

```bash
# 패키지 설치
npm install

# 개발 서버 실행
npm run dev
```

http://localhost:3000 에서 확인

## 기술 스택

- Next.js 16
- Tailwind CSS
- lightweight-charts
